# 名字竞技场 但是 rust

[https://deepmess.com/namerena/](https://deepmess.com/namerena/)

## 用法:

```text
Usage: namerena-rs.exe [OPTIONS] --team <TEAM>

Options:
      --start <START>                [default: 0]
      --end <END>                    [default: 18446744073709551615]
  -t, --thread-count <THREAD_COUNT>  [default: 10]
      --top <TOP>                    [default: 750]
      --team <TEAM>
  -h, --help                         Print help
```
